Metodos http
    GET Leitura
    POST Criação
    PUT Atualização
    DELETE Deleção
    PATH Atualização parcial

Códigos
    1XX: Informativo
    2XX: Confirmação
        200 - Requisiação bem sucedida
        201 - Created
    3XX: Redirecionamento
        301: Moved Permanently
        302: Moved
    4XX: Erros do cliente
        400: Bad request
        404: Not found
    5XX: Erros no servidor
        500: Internal server error
        502: Bad Gateway

Corpo
    Header params
    Query params
    Route Params
    Body Params
